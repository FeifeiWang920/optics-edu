## 3. 内容架构

### 3.1 理论核心（左栏）

**费马原理的数学表述**
```
δ∫n(s)ds = 0
```
- **光程（OPL）**: L = ∫n ds = c·t，等效真空距离
- **Stationary**: 不仅是极小值，也可能是极大值或拐点（椭圆焦点反射）
- **程函方程**: (∇S)² = n²，几何光学的波动根源

**从费马原理推导反射/折射定律**
- 使用欧拉-拉格朗日方程的变分推导
- 展示经验定律的必然性基础

**物理意义的深层理解**
- 与费曼路径积分的量子力学联系
- λ→0 时波动光学退化为几何光学

### 3.2 交互式演示（右栏）

**组件1: 光程对比器**
- 可拖动起点A和终点B
- 显示：实际光路 vs 直线路径的光程对比
- 实时计算并高亮显示光程stationary的路径

**组件2: 折射路径求解器**
- 给定两种介质 n₁, n₂
- 用户调节入射点位置
- 实时显示满足斯涅尔定律的路径（费马原理的必然结果）

### 3.3 车灯光学应用

**自由曲面设计的理论根基**
- 逆向费马原理：从目标配光反推表面形状
- 光线追迹的数值实现
