# 色度学深度解析页面设计文档

## 项目信息

- **创建日期**: 2026-03-24
- **设计目标**: 在现有色度学页面基础上，新增深度解析页面，系统讲解色度学从入门到提高的内容
- **参考来源**: 费曼物理学讲义第35章、CIE 15:2018标准、Wyszecki《Color Science》、Hunt《The Reproduction of Colour》

## 页面定位

- **路由**: `/color/advanced`
- **入口**: 从当前色度学页面"高级概念"区域添加进入链接
- **目标受众**: 混合（入门学习者到专业工程师）
- **结构**: 分章节卡片式，可跳转的模块化学习

## 章节结构（共8章）

### 第1章：从彩虹说起
**引用**: 费曼物理学讲义 35-1 "The human eye"
**难度**: 入门级

**内容**:
- 光的物理本质：电磁波谱、可见光范围（380-780nm）
- 彩虹的形成：折射、色散
- 为什么天空是蓝色的（瑞利散射）

**交互组件**: `SpectrumSlider`（复用现有）

**可展开数学**:
```
瑞利散射强度 ∝ 1/λ⁴
天空颜色 = ∫ I(λ) · P(θ,λ) · V(λ) dλ
```

---

### 第2章：人眼的精密设计
**引用**: 费曼物理学讲义 35-2 "Color depends on intensity"
**难度**: 入门级

**内容**:
- 视网膜结构：视杆细胞与视锥细胞分布
- LMS三种视锥的光谱灵敏度
- 暗视觉与明视觉（Purkinje效应）

**交互组件**: `LMSSensitivityChart`（新增）

**可展开数学**:
```
L = ∫ S(λ) · l̄(λ) dλ
M = ∫ S(λ) · m̄(λ) dλ
S = ∫ S(λ) · s̄(λ) dλ
```

---

### 第3章：颜色匹配实验
**引用**: 费曼物理学讲义 35-3 "The mixing of colors"
**难度**: 中级

**内容**:
- 格拉斯曼定律（Grassmann's Laws）
- Wright & Guild的RGB色匹配实验
- 负色匹配值的物理意义

**交互组件**: `ColorMixer`（复用现有）

**可展开数学**:
```
C ≡ R·[R] + G·[G] + B·[B]

当C为单色光时，某些波长需要"负"的原色：
C + (-R)·[R] ≡ G·[G] + B·[B]

这促成了虚拟原色X,Y,Z的诞生
```

---

### 第4章：CIE 1931标准系统
**引用**: CIE 15:2018 核心章节
**难度**: 中级

**内容**:
- RGB → XYZ转换的矩阵推导
- CIE 1931标准观察者的定义
- 色匹配函数 x̄(λ), ȳ(λ), z̄(λ)

**交互组件**: `ColorMatchingCurves`（新增）+ `CIE1931Explorer`（复用）

**可展开数学**:
```
[X]   [ 2.7689  1.7517  1.1302 ]   [R]
[Y] = [ 1.0000  4.5907  0.0601 ] · [G]
[Z]   [ 0.0000  0.0565  5.5943 ]   [B]

色品坐标：
x = X/(X+Y+Z), y = Y/(X+Y+Z)
```

---

### 第5章：读懂色品图
**引用**: Hunt《The Reproduction of Colour》
**难度**: 中级

**内容**:
- 马蹄形边界的物理意义（光谱轨迹）
- 等能白点（E点）与主波长
- 互补色与饱和度的几何解释

**交互组件**: `CIE1931Explorer`（复用现有库）

**可展开数学**:
```
主波长计算：从白点向样品点作延长线
与光谱轨迹的交点即为该颜色的主波长

饱和度 = (样品到白点的距离) / (光谱轨迹点到白点的距离)
```

---

### 第6章：均匀色空间与色差 ⭐新增
**引用**: CIE 15:2018 Section 8, CIE 142-2001
**难度**: 高级

**内容**:

**6.1 CIE 1931的非均匀性问题**
- xy色品图的距离 ≠ 视觉感知差异
- MacAdam椭圆：相同色差在色品图上呈现不同大小的椭圆

**6.2 均匀色空间的发展**
- CIE 1960 UCS色度图（u, v坐标）
- CIE 1976 UCS色度图（u', v'坐标）
- CIE 1976 L*a*b*（CIELAB）色空间

**6.3 色差公式演进**
- ΔE*ab（CIE 1976）：欧氏距离，简单但不够精确
- ΔE*94（CIE 1994）：考虑明度、彩度、色调权重
- ΔE*00（CIE 2000）：当前最精确，考虑非对称性

**6.4 车灯光学中的色差容限**
- 同色异谱指数的工业应用

**交互组件**:
- `ColorDifferenceCalculator`（新增）
- `MacAdamEllipses`（新增）

**可展开数学**:
```javascript
// CIE 1976 UCS 坐标转换
u' = 4X / (X + 15Y + 3Z)
v' = 9Y / (X + 15Y + 3Z)

// CIELAB 空间
L* = 116·f(Y/Yn) - 16
a* = 500·[f(X/Xn) - f(Y/Yn)]
b* = 200·[f(Y/Yn) - f(Z/Zn)]

// CIE 2000 色差公式
ΔE*00 = √((ΔL'/(kL·SL))² + (ΔC'/(kC·SC))² +
        (ΔH'/(kH·SH))² + RT·ΔC'·ΔH'/(kC·kH·SC·SH))
```

---

### 第7章：同色异谱与色适应
**引用**: Wyszecki & Stiles《Color Science》
**难度**: 高级

**内容**:
- 同色异谱的定义与工程意义
- 色适应变换（von Kries变换）
- 不同光源下的颜色恒常性

**交互组件**: `MetamerismDemo`（新增）

**可展开数学**:
```
von Kries色适应：
[L']   [1/L_w  0     0   ]   [L]
[M'] = [ 0    1/M_w  0   ] · [M]
[S']   [ 0     0    1/S_w]   [S]
```

---

### 第8章：车灯中的色度学实践
**引用**: ECE R48, SAE J578等法规
**难度**: 应用级

**内容**:
- ECE色度区域的定义与解读
- 车灯颜色检验的实际流程
- LED与卤素灯的色度特性对比

**交互组件**: 法规色度区域叠加显示

**可展开数学**:
```
色度容差计算：
在CIE 1976 UCS色度图中：
ΔE = √((u'₁-u'₂)² + (v'₁-v'₂)²)

ECE法规通常规定容差范围（如±0.01）
```

## 设计决策记录

1. **章节顺序**: 遵循"物理→生理→心理→应用"的认知逻辑
2. **色差章节位置**: 放在色品图之后，因为需要CIELAB色空间基础
3. **数学处理**: 使用可展开区域，不强制展示所有推导
4. **复用优先**: 尽可能复用现有组件，降低实现成本
